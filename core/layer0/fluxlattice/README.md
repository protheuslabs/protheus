# FluxLattice

Internal systems tool for settle/morph/flux primitives.

## CLI

```bash
cargo run --manifest-path core/layer0/fluxlattice/Cargo.toml --bin fluxlattice -- init
cargo run --manifest-path core/layer0/fluxlattice/Cargo.toml --bin fluxlattice -- settle
cargo run --manifest-path core/layer0/fluxlattice/Cargo.toml --bin fluxlattice -- morph dynamic
cargo run --manifest-path core/layer0/fluxlattice/Cargo.toml --bin fluxlattice -- status
```
