# Snowball Engine

Thin shell over core `snowball-plane`.

Examples:

- `protheus snowball start --cycle-id=s17 --drops=core-hardening,app-refine`
- `protheus snowball melt-refine --cycle-id=s17 --regression-pass=1`
- `protheus snowball compact --cycle-id=s17`
- `protheus snowball backlog-pack --cycle-id=s17`
- `protheus snowball status --cycle-id=s17`
