# Comment Mapper Demo

Internal toolkit demo for stream-of-thought mapping with optional intercept controls.

Run:

```bash
node apps/comment-matrix/run.js
```

This executes:

```bash
node client/systems/ops/protheusctl.js toolkit comment-mapper --persona=vikram_menon --query="Should we prioritize memory or security first?" --gap=1 --active=1
```
