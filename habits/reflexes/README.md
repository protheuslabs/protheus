# Habits Reflexes

This directory holds reflex-layer policy for fast micro-routine promotion from habits.

- Runtime routines are stored in `state/adaptive/reflex/routines.json`.
- Use `node habits/scripts/reflex_habit_bridge.js sync` to promote eligible habits.
- Use `node habits/scripts/reflex_habit_bridge.js gc` to degrade/disable stale reflex routines.
