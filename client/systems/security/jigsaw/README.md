# Project Jigsaw

AttackCinema capture, replay, and deterministic highlight generation for security incidents.

## Commands

- `node client/systems/security/jigsaw/attackcinema_replay_theater.js capture --severity=<low|medium|high|critical> --source=<source>`
- `node client/systems/security/jigsaw/attackcinema_replay_theater.js replay --clearance=4 [--incident-id=<id>]`
- `node client/systems/security/jigsaw/attackcinema_replay_theater.js status`
