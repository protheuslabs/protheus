# UI Accessibility & Interaction Contract

`V4-FORT-001` accessibility contract.

## Accessibility

- keyboard navigation must cover primary actions
- focus indicators must remain visible
- contrast checks required for critical text
- aria labels required when semantic labels are insufficient

## Interaction

- command palette discoverability must be documented
- responsive behavior requirements apply to desktop/tablet/mobile
- state transitions (`hover`, `focus`, `active`, `disabled`) must be explicit
