# UI Design Token Standard

`V4-FORT-001` design token baseline.

## Token Families

- color tokens: semantic (`surface`, `text`, `accent`, `danger`)
- spacing tokens: scale-based (`xs`, `sm`, `md`, `lg`, `xl`)
- typography tokens: heading/body/mono emphasis roles
- motion tokens: `fast`, `normal`, `slow` transitions

## Rules

- New UI surfaces must consume semantic tokens instead of hardcoded values.
- Token updates require changelog note and accessibility re-check.
