## Current Focus

### Kubernetes Guardian 🛡️ 
*Open source deployment safety toolkit*

Admitting you have a deployment problem is the first step. Guardian enforces organization-wide safety policies before manifests hit the cluster—no more 3 AM pages because someone forgot resource limits.

**Stack:** Go, Kubernetes admission webhooks, Prometheus  
**Status:** v0.1.0 in development, 57 commits, active community interest

### Observability Patterns 📊
*Developer tooling for better production visibility*

Building reusable patterns for distributed tracing, structured logging, and incident correlation. Because you can't fix what you can't see, and "it's slow" is not a debuggable bug report.

**Stack:** OpenTelemetry, Grafana, custom exporters
