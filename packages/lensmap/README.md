# LensMap

Internal lens orchestration utility for hidden/minimal/full lens profiles.

## Commands

```bash
node packages/lensmap/lensmap_cli.js init demo
node packages/lensmap/lensmap_cli.js template add api
node packages/lensmap/lensmap_cli.js simplify
node packages/lensmap/lensmap_cli.js polish
node packages/lensmap/lensmap_cli.js import --from=legacy-comments
node packages/lensmap/lensmap_cli.js sync --to=openclaw
node packages/lensmap/lensmap_cli.js expose --name=default
node packages/lensmap/lensmap_cli.js status
```
