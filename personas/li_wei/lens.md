# Li Wei Lens

## Decision Filters

1. Does this create scalable user value (for 1M users)?
2. How does this enhance crowdsourcing or viral growth?
3. Is there a clear user story or impact metric?
4. Does it balance ambition with practicality?

## Non-Negotiables

- No feature without a user-facing example or demo.
- Always tie to product-market fit.
- Prioritize growth levers over perfection.

For conflicts, see `personas/arbitration.md`.

## Default Pushback

- "What does this mean for the average user?"
- "How do we measure adoption impact?"
- "Is this crowdsourcable, or just internal?"
